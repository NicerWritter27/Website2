If you are Opera, go to Method 1, if you are using chrome or other browsers, then go to Method 2

Method 1

Download on Opera

(if you are using Opera, Opera GX, Opera Dev, etc. then this is the easiest way)

Step 1

Go to Opera's addons store

Visit Opera addons

Step 2

Click the green button that says "Add to Opera" to install Pacman popup

Step 3

A popup should appear at the top right saying "Pacman popup has been added to Opera", then if you want to pin it (next to the address bar), click the "cube" icon at the top right, then click on the "pin" icon next to Pacman popup's section

WARNING: If you install Pacman popup from Opera's addons store, then Pacman popup's icon will also appear on Opera's sidebar, you can also open it there, but it is in beta version, so bugs and glitches may appear

Step 3

Look for the "developer mode" switch/option, turn it on, then "load unpacked" and "pack extension" buttons should appear, you can disable "developer mode" later

Step 4

Click "load unpacked", search for the extracted folder (folder from the extracted file from the zip file you downlaoded earlier), select it

Step 5

If this is your first time to downlaod extensions to your browser, then you can find your installed extensions when you click the "puzzle" icon on the top right (next to the address bar), all your installed extensions will appear, to play the game, click on it's icon, you can also pin it by clicking the pin icon next to it, but in Edge, it will automatically be pinned
